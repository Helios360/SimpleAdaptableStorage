# NewGenCRM – Next Generation Stack

This branch is a special milestone for the project.

The goal is to rebuild and structure the system using a **modern, next-generation stack** designed for performance, modularity, and long-term scalability.

## Tech Stack

The project will be implemented using:

- **Bun** – fast JavaScript runtime and toolkit  
- **SvelteKit** – modern full-stack framework for building reactive applications  
- **Prisma + PostgreSQL** – type-safe ORM with a robust relational database

## Why This Stack

Adopting this stack allows the project to become:

- **More modular** – easier to extend and maintain  
- **Easier to develop** – better tooling and developer experience  
- **Fast and future-proof** – modern runtime and architecture  
- **Easy to scale** – solid database and backend structure

## Vision

**NewGenCRM** is not just a CRM for a single enterprise.

It is intended to be a **modular CRM platform** that combines:

- High performance  
- Strong UI/UX design  
- The reliability of traditional enterprise applications  

## Cross-Platform Direction

To make the platform **multiplatform and highly adaptable**, the plan is to package the application using **Tauri**, enabling native builds while keeping the flexibility of the web stack.
